Webbens historia och teknikutveckling

Fråga 1:

  • 1966: Världens första dokument med hyperlänkar skapades av Douglas Engelbart och Jeff Rulifson. Hyperlänkar var det första effektiva metoden för att navigera dokument
  • 1969: Världens första webbprotokoll skapades på ARPNET, IP-protokollet. Det utvecklades av USA för att koppla samman datanätverk under kalla kriget.
  • 1990: I CERN-labratoriet i Genéve utvecklade Tim Berners-Lee World Wide Web genom att kombinera internet med hypertext. World wide web möjliggjorde enkel delning av information och kommunikation.
  • 1991: Tim Berners-Lee skapade världens första webbläsare och publicerade år 1991 världens första hemsida online tillsammans med sidbeskrivningsspråket HTML som gorde det lätt att ändra webbplatsens struktur.
  • 1994: Håkon Wium Lee utvecklade css som med hjälp av stylesheets förändrade hemsidors utseende
  • 1999: Web 2.0 utvecklades av Darcy DiNucci och meförde interkativa webbsidor.

Fråga 2:

När du skriver in www.google.com i din webbläsare så händer följande:
  • 1 - Webbläsaren skickar en förfrågan till en DNS server som översätter domännamnet till en ip adress utifrån ett register.
  • 2 - Med hjälp av TCP protokollet skapar datorn en förbindelse till googles server. Webbplatser med stöd för HTTPS/SSL/TLS krypterar trafiken.
  • 3 - En HTTP förfrågan skickas som beskriver vad servern ska svara med.
  • 4 - Servern hämtar datan och bygger svaret utifrån HTTP förfrågan och skickar sedan tillbaka svaret till webbläsaren.
  • 5 - Webbläsaren tolkar serverns svar om hur den ska framställa datan. Medan webbläsaren framstället HTML innehållet skickas fler förfrågningar om andra resurser, såsom css, bilder och javascript.
